# RehaNote

## サイト概要
### サイトテーマ
リハビリ・介護職向けに、軽運動と記録文例を管理・共有できるコミュニティサイト
​
### テーマを選んだ理由

私は作業療法士としてリハビリ職に従事しています。  
日々の業務では、各職員が考えたリハビリメニュー（主に運動）を利用者に指導・時に一緒に実施し、それを記録しています。  
業務を行う中で、「運動内容や記録が属人化しやすい」「効果的な運動が共有されにくい」といった課題を感じてきました。また、「利用者に合わせた運動を考える」「記録文をゼロから作成する」という作業には、毎回多くの時間と労力がかかります。  
そこで、「運動内容」を共有しさらに「記録文例」とセットで扱う仕組みを作ることで、現場業務効率化だけでなくリハビリの質向上にもつながるのではないかと考え、本ポートフォリオを制作しました。
​
### ターゲットユーザ
* リハビリ職（理学療法士・作業療法士）
* 介護士、看護師、高齢者施設勤務者など軽運動を提供する機会のある人
​
### 主な利用シーン
* 利用者に合わせた運動を考える時
* 記録文をゼロから作成する時
